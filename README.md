# Sistema de Dimensionamento para Comunicação Visual

Aplicativo profissional para calcular automaticamente materiais de comunicação visual a partir de PDFs.

## Funcionalidades
- Leitura automática de PDF
- Extração de medidas (m², largura x altura, etc.)
- Cálculo de materiais (adesivos, lonas, ACM, letras, etc.)
- Relatório profissional
- Exportação CSV

## Como usar
1. Faça upload do PDF da Ordem de Produção
2. Aguarde o processamento automático
3. Visualize os materiais calculados
4. Exporte o relatório
